# MVP Kapsamı (v0)

## Amaç (Önce neyi öğrenmek istiyoruz?)
Kullanıcı (öğrenci), test sonucunu girip önerilen bir sonraki testi görmek için
uygulamayı haftada en az 3 kez açıyor mu?

## MVP İçinde Olanlar
1) Test Sonucu Girişi
- Ders seçimi (ör. Matematik)
- Konu bazlı doğru / yanlış ya da net girişi
- Kaydet

2) Konu Bazlı Anlık Analiz
- Konular: Zayıf / Orta / İyi
- Basit bir liste veya tablo yeterli

3) Önerilen Sonraki Test / Konu
- Zayıf konulara göre sıralı öneri listesi
- “Önce şunu çöz”, “sonra bunu çöz” gibi

## MVP Dışında Olanlar (Şimdilik YOK)
- Öğretmen / okul paneli
- Kullanıcı hesabı / giriş
- Karmaşık grafikler
- Makine öğrenmesi
- Detaylı geçmiş analizleri
- Büyük veri tabanı tasarımı

## Başarı Ölçütleri (MVP)
- Kullanıcı başına haftalık test girişi sayısı
- Test girdikten sonra öneri ekranını gören oran
- 7 gün içinde geri gelen kullanıcı oranı
